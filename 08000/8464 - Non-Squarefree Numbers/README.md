## 문제

A positive integer k is called squarefree if k is not divisible by $d^{2}$
for any d > 1.

## 입력

The input contains one integer n, 1 ≤ n ≤ $10^{10}$
.

## 출력

Your program should output one integer, the n-th non-squarefree number.

## 예제

### 예제 입력 1

```
10
```

### 예제 출력 1

```
27
```
