## 문제

수 N개 $A_{1}$
, $A_{2}$
, ..., $A_{N}$
이 주어진다. 이때, 연속된 부분 구간의 합이 M으로 나누어 떨어지는 구간의 개수를 구하는 프로그램을 작성하시오.

즉, $A_{i}$ + ... + $A_{j}$
(i ≤ j) 의 합이 M으로 나누어 떨어지는 (i, j) 쌍의 개수를 구해야 한다.

## 입력

첫째 줄에 N과 M이 주어진다. (1 ≤ N ≤ $10^{6}$
, 2 ≤ M ≤ $10^{3}$
)

둘째 줄에 N개의 수 $A_{1}$
, $A_{2}$
, ..., $A_{N}$
이 주어진다. (0 ≤ $A_{i}$
≤ $10^{9}$
)

## 출력

첫째 줄에 연속된 부분 구간의 합이 M으로 나누어 떨어지는 구간의 개수를 출력한다.

## 예제

### 예제 입력 1

```
5 3
1 2 3 1 2
```

### 예제 출력 1

```
7
```
