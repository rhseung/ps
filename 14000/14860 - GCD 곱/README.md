## 문제

N과 M 이 주어졌을 때, GCD(1, 1) × GCD(1, 2) × ... × GCD(1, M) × GCD(2, 1) × GCD(2, 2) × ... × GCD(2, M) × ... × GCD(N, 1) × GCD(N, 2) × ... × GCD(N, M)을 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 N과 M이 주어진다. (1 ≤ N, M ≤ 15,000,000)

## 출력

첫째 줄에 곱을 $10^{9}$ +7로 나눈 나머지를 출력한다.

## 예제

### 예제 입력 1

```
4 4
```

### 예제 출력 1

```
96
```
