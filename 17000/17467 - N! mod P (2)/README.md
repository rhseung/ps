## 문제

양의 정수 N과, N보다 큰 소수 P가 주어질 때, N!을 P로 나눈 나머지를 구하여라.

## 입력

첫째 줄에 N과 P가 공백으로 구분되어 주어진다.

## 출력

N!을 P로 나눈 나머지를 구하여라.

## 제한

- 1 ≤ N < P ≤ $10^{9}$
- P는 소수

## 예제

### 예제 입력 1

```
4 7
```

### 예제 출력 1

```
3
```

## 예제

### 예제 입력 2

```
999999936 999999937
```

### 예제 출력 2

```
999999936
```
